# CalorieCalculator
## 問題1

栄養素の中でも以下の栄養素は「三大栄養素」と言われています。また、英語の頭文字をとって「PFC」とも呼ばれています。

- **Protein (たんぱく質)**
- **Fat (脂質)**
- **Carbohydrate (炭水化物)**

三大栄養素はからだの土台となるものでエネルギー源にもなります。それぞれ以下の変換式をもとに、摂取量(g)からエネルギー源であるカロリー(kcal)の計算が可能です。

- **Protein (たんぱく質)**: 1(g) = 4(kcal)
- **Fat (脂質)**: 1(g) = 9(kcal)
- **Carbohydrate (炭水化物)**: 1(g) = 4(kcal)

「三大栄養素」それぞれについて、摂取量(g)からカロリー(kcal)を計算するクラスを作成してください。尚、桁数についてはそれぞれ以下の通りとします。

- **摂取量(g)**: 小数第一位 (小数第二位を四捨五入)
- **カロリー(kcal)**: 整数 (小数第一位を四捨五入)

## 問題2

「問題1」で作成したクラスを使用して、以下の「納豆ご飯」の総カロリー(kcal)を計算するコードを書いてください。

- **白米**
  - P: 2.5g
  - F: 0.3g
  - C: 37.1g
- **納豆**
  - P: 16.5g
  - F: 10.0g
  - C: 12.1g

## 問題3

「問題1, 2」を通して作成したクラスの簡単なテストコードを書いてください。また、可能な限りテスト観点についてコメントに記載してください。コーディングテスト終了後に、どのような観点や意図を持っているかをお聞きいたします。
